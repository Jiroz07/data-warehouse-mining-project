# Data Warehouse & Data Mining Project

This project focuses on the design and implementation of a Data Warehouse and data mining processes using SQL Server Integration Services (SSIS).

The solution implements ETL workflows to extract, transform, and load data from source systems into a structured data warehouse for analytical purposes.

## Technologies
- SQL Server
- SQL Server Integration Services (SSIS)
- SQL Server Data Tools (SSDT)
- ETL Processes

## Project Type
Academic / Practice project

## Focus
- Data integration
- ETL design
- Data Warehouse concepts
- Analytical data preparatiomn
